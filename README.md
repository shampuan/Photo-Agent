# Photo-Agent
Duplicate Photograph Finder and Cleaner


It finds multiple image files on your computer and lists them for you to mark. You simply delete the ones you don't want. It's that simple!

The program is minimal in size and has very few dependencies (basic Python and Qt5 libraries).
